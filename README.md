# OpenCV Projects

## Overview
This repository contains projects and implementations using OpenCV for computer vision and image processing applications. The project focuses on image analysis, object detection, video processing, and real-time computer vision techniques using Python.

## Features
- Image processing and manipulation
- Object detection
- Face detection and recognition
- Video processing
- Edge and contour detection
- Real-time camera applications
- Computer vision model integration

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
- TensorFlow
- Jupyter Notebook

## Installation

Clone the repository:

```bash
git clone https://github.com/DrushtiLadhani/OpenCV.git
cd OpenCV
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the notebook or Python scripts:

```bash
jupyter notebook
```

or

```bash
python main.py
```
